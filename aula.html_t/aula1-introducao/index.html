O CSS

Folha de estilo em cascata, serve para fornartar as TAGS definidas no HTML.

As formatações incluem cor ,alinhamento ,altura,largura,aparência de objeto em geral.

Principais TAGS

  • A TAG <p>, serve para definir um parágrafo
  • A TAG <h1>, serve para definir Titulo
  • A TAG <br>, serve para quebrar linha
  • TAG <code>, serve para definir area de código
  • TAG <ul>, serve para criar lista não ordenadas
